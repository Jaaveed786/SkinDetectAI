Ultimate Deployment & Setup Instructions

1 Install Mandatory Prerequisites

Ensure your system has the following software installed before proceeding:

  • Python 3.9+: Download Here
  • Node.js 20+ (LTS): Download Here

2 Configuration

Before running, you must set up your environment variables:

  1. Go to web-app/, rename .env.example to .env.
  2. Go to backend/, rename .env.example to .env.
  3. Open backend/.env and configure your JWT_SECRET_KEY and API keys.

3 Running the Application

Simply double-click the run.bat file in the root directory. This script will:

  • Create a Python virtual environment.
  • Install all backend dependencies (using requirements.txt).
  • Install all frontend dependencies (using npm install).
  • Start both servers simultaneously.

Once running, access the portals at:

Frontend: http://localhost:5173/SkinDetectAI/
Backend API: http://localhost:8000
